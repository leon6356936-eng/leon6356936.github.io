臨床/研究小工具

這裡會收錄、分享與展示我自己開發的，臨床與研究小程式。相關資源供大家參考。

圖一：實驗指導語介面

華語韻母鼻音化知覺測驗 (AXB Task)

本實驗旨在測試聽者對於華語韻母中「有/無鼻音」對比的辨析能力。這對於區分聲隨韻母（如：ㄢ、ㄣ、ㄤ、ㄥ）與單韻母的聲學特徵具有重要的臨床與教學意義。

實驗設計：採用標準 AXB 範式，受試者將聽到三聲語音，並需判斷中間的語音 (X) 較像第一聲 (A) 還是第三聲 (B)。

圖二：PsychoPy 實驗流程與迴圈設定

開放資源下載 (Open Resources)

目前線上互動版本尚在技術調整中，若您是 SLP 同儕或研究人員，歡迎下載原始實驗檔案直接在本地端的 PsychoPy 執行。

  • 📁 AxB_Nasal_Task.psyexp (實驗主程式)
  • 📊 conditions.xlsx (題庫與資源清單)
  • 🔊 Resource_Audio.zip (ba/ban 刺激音檔)

★ 本實驗設計版權屬於研究團隊，轉載或修改使用請註明出處。

圖三：華語母音 F1/F2 聲學空間分布

華語母音聲學空間自動化分析 (Praat & Python)

除了知覺測驗，聲學特徵的量化也是 SLP 臨床研究的重要環節。本工具結合 Praat Script 與 Python Data Analysis，能自動化提取母音的第一 (F1) 與第二共振峰 (F2)。

應用價值：透過建立母音三角形 (Vowel Triangle)，我們能更精確地觀察構音障礙患者在治療前、後的口腔運動空間變化。

  • 📜 Auto_Formant_Extract.praat (自動化提取腳本)
  • 🐍 Vowel_Space_Plot.py (Python 視覺化代碼)
